Xande Flix

Aplicação web para buscar detalhes de séries e filmes.

Deploy
https://xandeflixxx.vercel.app

Tecnologias
- HTML, CSS e JavaScript
- API: [TVMaze](https://www.tvmaze.com/api)
- PWA com Service Worker

Funcionalidades
- Busca de séries e filmes por texto
- Busca por voz (microfone)
- Funciona offline (PWA)
- Instalável na tela inicial do celular

Como usar
1. Digite o nome de uma série ou filme
2. Clique em **Buscar** ou pressione **Enter**
3. Ou clique no 🎙️ e fale o nome
